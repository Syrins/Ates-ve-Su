
# Ateş ve Su

Ateş ve Su oyunu online oynayabilirsiniz. 
Artık yan yana olmanıza gerek yok.


## Bölüm Editor

#### LevelGeneration Açınız

```text
  Bu kısımdan kendi bölümlerinizi yapabilirsiniz
```

| HARİTA TUŞU | Açıklama                |
| :-------- |  | :------------------------- |
| `G`  |kapı düğmesi ekler |
| `B`  |köprü düğmesi ekler |
| `M`  |orta blok ekler |
| `L`  |sol blok ekler |
| `R`  |sağ blok ekler |
| `W`  |küçük su ekler |
| `F`  |küçük lav ekler |
| `f`  |sol lav ekler |
| `i`  |orta lav ekler |
| `r`  |sağ lav ekler |
| `w`  |sol su ekler|
| `a`  |orta su ekler|
| `t`  |Sağ su ekler|

```text
  1. Level Tasarımı
```
